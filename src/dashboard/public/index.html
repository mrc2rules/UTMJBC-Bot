AskOctavia
Configure the /askai Gemini chatbot that answers member questions about UTM.
Chatbot AI Model
Gemini model used for the /askai slash command. Independent of the scraper model.

Recommended: gemini-2.5-flash (fast) · gemini-2.5-pro (higher quality)

How AskOctavia Works
  • Members run /askai <question> in any channel.
  • The question is sent to the configured Gemini model with UTM context.
  • The response is posted as a formatted embed in the same channel.
  • All responses carry a disclaimer that AI answers may not be accurate.
